## 安装&更新
在Yunzai根目录运行命令
```
git clone https://gitee.com/Tloml-Starry/Button ./plugins/Button/
```

如何更新？
 - 使用Yunzai自带指令`#全部更新`
 - 运行`UPDATE.bat`文件，位置在"Yunzai/plugins/Button"内
